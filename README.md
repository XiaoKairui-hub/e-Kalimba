# e-Kalimba(thumb-piano)
This program is actually the final project of SUSTech **EE351** microsystems course. Inspired by this [interesting gadget](https://github.com/LiferLifer/EE351-eMeritBox), also aiming at making something fun, I made this bluetooth+local bi-functional **e-Kalimba** (right) based on Raspberry Pi 4b (course requirement) and ESP32.
